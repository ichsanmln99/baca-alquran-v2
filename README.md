## Installation

```sh

git clone https://github.com/ichsanmln99/baca-alquran-v2.git

cd baca-alquran-v2

npm install

# development
npm run dev

# production
npm run build

# preview
npm run preview

```

Development akan berjalan pada `http://localhost:3000`
